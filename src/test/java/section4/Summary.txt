// stepDefinition file
@When("user login into application with {string} and password {string}") // {string} is a parameter type is selenium
	public void user_login_into_application(String username, String password) {
		System.out.println("Username is:- "+username+" and password is:- "+password);
	}

[Regex pattern in step definition file]
----------
Regex PatternSyntax is:-
========================
	1. ["^user login into application with (.+) and password (.+)$"]
		1.1) starts with ^
		1.2) ends with $
		1.3) variable dataTypes are denoted in the step file like this (.+)
		1.4) then we can write data - without "String" format also.
